    versioning-add

    @mixin versioning-add($name, $name, $name) { ... }

    Description

    Logic for adding versioning to the global list $au-glob-dependencies

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $name

    Name of the module adding this dependency

    		String—none
    $name

    Version of the module adding this dependency

    		String—none
    $name

    Dependencies of this module

    		List—none

    Output

    The variable $au-glob-dependencies will be populated with the items that have been passed

    Example

    $name: "accordion";
$version: "1.0.0";
$dependencies: (
	("core", "2.0.0"),
	("grid", "1.0.0"),
);

@include versioning-add( $name, $version, $dependencies );
